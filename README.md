# Verilog Digital Design Projects

This repository contains basic digital logic designs implemented using Verilog HDL.

## Contents
- Combinational Circuits: Adders, Multiplexers, Decoders
- Sequential Circuits: Flip-Flops, Counters, Registers
- Testbenches for functional verification

## Tools Used
- Verilog HDL
- ModelSim
- Quartus Prime

## How to Simulate
1. Compile the Verilog files in ModelSim
2. Run the testbench
3. Verify waveform outputs and imolement in FPGA
